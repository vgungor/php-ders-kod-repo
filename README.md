# MBP192 - PHP Programlama Örnekleri

Bu depo, **MBP192** kodlu ders kapsamında işlenen PHP (*Hypertext Preprocessor*) programlama diline ait temel ve orta seviye uygulama örneklerini içermektedir. İçerik, web geliştirme sürecinde PHP'nin dinamik yapısını, form yönetimini ve temel algoritma mantığını anlamaya yönelik pratik kod parçaları sunar.
